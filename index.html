HTML

¿Qué es HTML?

No es la parte bonita de la web, eso se hace con otro lenguaje, con html creamos los pilares de la página, toda su estructura.


Para comprender esto, utilizaremos una analogía; somos unos arquitectos y vamos a construir una casa. Iniciamos colocando las columnas, paredes, techo, espacio para puertas y ventanas. En programación esto es HTML.


Pero aún le falta mucho, el estilo de las puertas y ventanas, el color de la casa, las plantas que adornan, los cuadros, los sillones, etc., eso lo hacemos con CSS.


Ahora que comprendes que es HTML, y que debes preocuparte solo por la estructura y no por su apariencia, podemos continuar.


Tipos de archivos

Todos los archivos de HTML, deben guardarse con la extensión .html para que sea reconocido como dicho archivo. Es para identificar el tipo de archivo; las imágenes son tipo JPG, PNG, los archivos de Excel son xls, los de Photoshop PSD y los de html son .html.


Archivo index.html

Los servidores tienen en su configuración interna mostrar como primer página el archivo.


Por ejemplo: Entras a tu navegador y buscas www.facebook.com, el sitio de Facebook tiene muchas páginas internas y seria una tarea imposible para el servidor saber que página seleccionar de entre todas para mostrartela.


Es por ello que se configura un index, para decirle al servidor esta es la págia que deseo que muestres cuando me busquen.


Sintaxis

En HTML existen reglas para escribir el código, las cuales debemos seguir para poder estructurar bien nuestro sitio web.


Es parecido a la sintaxis cuando hablamos, no lo hacemos de manera desordena, siempre lleva una lógica y estructura


Semántica

En html regula el abuso de ciertas etiquetas y se penalizan con el posicionamiento web (SEO).


TAGS/Etiquetas

Las etiquetas son elementos de HTML con los cuales podemos ordenar el contenido del sitio web.


La apertura de una etiqueta se realiza con estos símbolo "< >".


Ejemplo de la apertura de una etiqueta <etiqueta>. La mayoria de las etiquetas deben cerrarse y para ello agreamos una barra inclinada "/", quedando la etiqueta cerrada de esta forma </etiqueta>.


¿Y como sabemos cuando cerrar una etiqueta?


Las etiquetas que se deben cerrar son las que tienen contenido dentro de ellas. Por ejempo: <p>esta es una etiqueta de parráfo</p>.


Atributos/Propiedades de las etiquetas

El atributo de una etiqueta, en palabras simples, son las carácteristicas que podemos modificar. Para establecer una atributo debemos definir entre comillas su valor.


<etiqueta atributo="valor"></etiqueta>


Ejemplo:


<input type="text">


En el ejemplo estamos definiendo una etiqueta de input(entrada) y un atributo type(tipo), a ese atributo se le asigno el valor text(texto).


Cabe mencionar que una etiqueta puede tener varios atributos, eso dependenderá de las necesidades que tengamos.

Tu primer "Hola Mundo"

Ya sabes los conceptos necesarios, para aventurarte en el Desarrollo Web, también conocido como Front End.


Para escribir tu primer hola mundo es necesario que descargues un editor de texto. Anteriormente te recomendamos utilizar VSCode, así que daremos por hecho que ya lo tienes instalado.


Abre tu editos de código y crea un nuevo documento HTML, al cual le pondrás por nombre index.html


Ahora que guardaste tu archivo, vas a escribir tu primer ¡Hola mundo!. Escribe el siguiente código (no lo copies y pegues).


Ejercicio #1


<!DOCTYPE html>
<html>
<head>
<title>Primera página web</title>
</head> <body>
<h1>¡Hola mundo!</h1>
</body>
</html>

Explicación del código


El !DOCTYPE indica que el documento es de tipo html en su última versión (v5).


Html, contiene toda la página, lo que se muestra al usuario como lo que no.


Head, acá programaremos todo aquello que el usuario no verá, a excepción de la etiqueta title. Acá es donde se colocan las configuraciones y enlaces de la página.


Title, muestra el título de la página en la pestaña del navegador. Justo en la parte superior.


Body, acá se programa todo lo que el usuario podrá ver. Botones, cajas, secciones, imagen, texto, video, etc.


H1, es una etiqueta de título nivel 1. Por página debe de existir solo 1, no puedes utilizar más, o puede afectar el posicionmineto SEO.


Para abrir la página que acabas de crear, bastará con ir al lugar dónde lo guardaste al inicio, das doble clic sobre el y listo, acabas de crear tu primer página web con un ¡Hola mundo!.


Tipos de rutas

Las rutas son ubicaciones a los cuales podemos redirigir, por ejemplo: www.facebook.com es una ruta.


Rutas Locales: son las que en el mismo lugar que se encuentra nuestra página web.


Ejemplo ruta local:
<a href="contacto.html">Contacto</a>


Ruta local dentro de otra carpeta:
<a href="carpeta/contacto.html">Contacto</a>


Rutas Externas: son las que redirigen fuera de nuestro sitio web.


Ejemplo ruta externa:
<a href="https://facebook.com">Facebook</a>


HTTPS: Prótocolo de transferencia de hiper texto, este método es el que sirve para enlazar sitios externos.


¿Cómo volver entre carpetas?
Bastará con colocar .. para volver.
Ejemplo: <a href="../index.html">Inicio</a>

Hacer un salto de línea:

Cuando se esta redactando un texto es necesario hacer saltos de líneas (dar enter), pero en programación no funciona solo dando un enter. En HTML tenemos la etiqueta <br>, para realizar este salto de línea.


Textos en negrita

Una función muy utilizada en textos es la de colocarlos en negrita, y esto lo podemos lograr en HTML con las siguientes etiquetas: <b>Texto</b> ó <strong>Texto</strong>.


Abrir enlaces en otra pestaña

Las etiquetas poseen atributos/propiedades que podemos modificar para que realicen acciones que deseamos, con la etiqueta <a href="https://google.com">Google</>, estamos utilizando el atributo href="", para idicarle la dirección.


Para decirle que abrá la dirección en otra página utilizaremos el atributo target="_blank", esto le indica al navegador que el link se abra en una nueva pestaña.


Ejemplo:
<a href="https://google.com" target="_blank">Google</a>

Multimedia

En HTML se pueden mostrar imagenes, videos y audios. Pareciera algo muy complicado de realizar, pero no es así, a continuación podrás comprenderlo.


Mostrar una imagen

Existe una etiqueta llamada <img>, dicha etiqueta no necesita ser cerrada, ya que el contenido a mostrar lo hace con un atributo y no entre la apertura y cierre.


<html>
<head>
<title>Mi primera imagen en HTML</title>
<body>
<img src="imagen.png" alt="codificando" width="300px">
</body>
</html>

Resultado:


Explicación del código:

La etiqueta para insertar imagen, necesita del atributo src="" (source/recurso) para hacer el llamado al archivo, este puede estar almacenado de forma local o externamente.


El atributo alt=""sirve para el Search Engine Optimatization (SEO), también para dispositivos que ayudan a personas con problemas de la visión, leen la descripción de alt="".


width="", se utiliza para darle un ancho especifico a la imagen, también va compañada de higth="", para definir el alto. 300px son los pixeles que ocupará la imagen en nuestro sitio web.


Seguramente te preguntas que pasa sino colocas el heigth="", la imagen tendrá el mismo valor de width para heigth, siendo esta una imagen cuadrada.


Cabe mencionar y recalcar que html es solo para la estructura, no para definir estilos, es por ello que si se modifica el tamaño de la imagen, debe realizarse con CSS. Pero para fines educativos es necesario que conozcas estas dos propiedades.


A la practica de moficar el estilo desde html, se le conoce como estilos en línea. Cuándo te adentrés en CSS comprenderas mejor esta definición.


Mostrar un video

A difernecia de la etiqueta para mostrar una imagen, la etiqueta para video si debe cerrarse quedando de la siguiente manera: <video src=""></video>. La ruta que se le pase al atributo src="" debe ser un video, con esto nos referimos a que el formato debe ser .avi .mp4 .mov etc.


Para terminar de configurar nuestro video, debemos hacer uso del atributo controls="", este atributo ya tiene un valor configurado por el navegador, así que entre las comillas no debe de ir nada.


<html>
<head>
<title>Mi primer video en HTML</title>
<body>
<video src="http://v2v.cc/~j/theora_testsuite/320x240.ogg" controls=""></video>
</body>
</html>

Formularios

Para crear un formulario utilizamos la etiqueta <form></form>, dentro ella se anida una etiqueta llamada <input type="">.


El input se refiere a una entrada de datos, este puede ser por ejempo un texto.


En informática existen varios procesos, pero los relacionados con el ipunt son: procesamiendo y salida.


Ejemplo contextualizado:

Imagina dos personas conversando, una ellas dice: Hola (input) la otra persona esucha (procesa) y dice hola como estas (output/salida).


Lo mismo que sucede en el ejemplo anterior: en tu página web hay un formulario con un espacio que dice envia tu comentario (input), se envia y llega al servidor (procesamiento), el servidor devuelve gracias por comentar (output/salida).

Tipo correo

Tipo color

Tipo fecha

<input type="date">



Tipo tiempo

<input type="time">



Tipo botón

<input type="button">



Tipo envió

<input type="submit">



Vamos a crear nuestro primer formulario

Para el ejemplo, vamos a utilizar los inputs de texto, correo, contraseña y enviar. Utilizaremos un atributo nuevo llamado placeholder="".


<html>
<head>
<title>Formulario</title>
<body>
<form>
<input type="text" placeholder="Nombre"><br>
<input type="email" placeholder="Correo"><br>
<input type="password" placeholder="Contraseña"><br>
<input type="submit" value="Enviar"><br>
</form>
</body>
</html>
