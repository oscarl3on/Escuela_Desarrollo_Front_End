Descubriendo HTML y etiquetas

HTML es el lenguaje informático que estructura todas las páginas web en Internet

El código HTML le dice a los navegadores web cómo mostrar contenido, imágnes, enlaces y texto en una página web.


<button> es una etiqueta de apertura. Le dice al navegador web que queremos comenzar a mostrar un botón.
